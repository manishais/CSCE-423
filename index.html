Accessibility Interests

I am developing an augmented reality (AR) system for phantom limb pain as my capstone project. The system provides real-time visual and sensory feedback to help amputees manage phantom pain by allowing them to see and interact with a virtual limb. Using motion tracking and haptic feedback, it builds on mirror therapy principles for a more personalized and immersive rehabilitation experience. I focus on accessibility and usability, ensuring the system adapts to different users. This project strengthens my passion for using technology to enhance accessibility and improve quality of life.
